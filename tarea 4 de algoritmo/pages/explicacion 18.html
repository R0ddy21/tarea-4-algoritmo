Explicacion Del Ejercicio 18

Declarar un arreglo con números

let numeros = [10, 15, 23, 30, 45, 50]

Usar un bucle for para iterar a través del arreglo desde el último elemento al primero.

for (let i = numeros.length - 1; i >= 0; i--) itera desde el último elemento al primero.

Usar una condición if para verificar si el número es par.

if (numeros[i] % 2 === 0) verifica si el número es par.
